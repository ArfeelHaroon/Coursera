HTML Entities

HTML entities are the reserved characters that are used in the HTML document. If we want to display these characters we use their entity. Its starts from &:

Example:

< - Less than

> - Greater than

& - Ampersand

© - Copywrite

 - nbsp stands for non-breaking space, meaning that strings separated by this entity will not be separated and put into separate lines.

Summery

HTML Entities

  • Help avoid rendering issues
  • Safeguard against more limited character encoding
  • Provide characters not available on a keyboard
