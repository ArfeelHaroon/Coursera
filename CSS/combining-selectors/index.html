Child Selector

Its a <p> the child of <article>. To select it we use CSS child selector. which is "article > p". It only effects the direct child.

Descendant Selector

This is a <p> inside a <section>, which is inside an <article>. The Descendant Selector selects all <p> elements inside <article>, regardless of depth. The CSS rule is "article p { }". It applies styles to any <p> inside <article>, even if nested within other elements.

Adjacent Sibling Selector (+)

The adjacent sibling selector selects an element that is immediately after a specified element. Example: "h2 + p { }" applies styles to the first <p> that comes right after an <h2>.

General Sibling Selector (~)

The general sibling selector selects all elements that are **siblings** of a specified element, not just the immediate one. Example: "h2 ~ p { }" applies styles to **all** <p> elements that follow an <h2>, regardless of position.
